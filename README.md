# davidchungnet 

The Personal Website for David G Chung

## Developing

### Built With
* HTML5 & CSS3  
* JavaScript  
* [ReactJS](https://reactjs.org/)  
* [GatsbyJS](https://www.gatsbyjs.org/)  
* GraphQL  

### Prerequisites
What is needed to set up the dev environment:
[npm](https://nodejs.org/en/)


### Setting up Dev

Here's a brief intro about what a developer must do in order to start developing
the project further:

```shell
git clone https://github.com/davidc4747/davidchungnet.git
cd davidchungnet/
npm install
npm run develop
```

### Building

```shell
npm run build

# Output will be stored inside the "public/" folder
```
